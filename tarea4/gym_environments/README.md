# Custom Gym Environments

- TwoArmedBandit-v0
- TwoArmedBandit-v1
- RobotBattery-v0
- FrozenLake-v2
- RobotMaze-v0

# Installation

```bash
pip install https://github.com/jesuspangulo/gym_environments/archive/refs/heads/main.zip
```

# Basic Usage

```python
import gym
import gym_environments

env = gym.make('TwoArmedBandit-v0')
```
